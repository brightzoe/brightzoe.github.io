为博客添加评论功能

2022年6月7日 · 阅读需 3 分钟
brightzoe

为本博客（docusaurus）添加 disqus 评论系统。

首先需要在 disqus 有一个账号，没有需要先注册。选择 "get started" =>" I want to install Disqus on my site" 。在 disqus 上配置对应的网站信息，配置的 Website Name 就是后面组件中要用到的 shortname。
