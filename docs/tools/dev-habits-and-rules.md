## 命名规则

**文件命名规则：**

- 文件夹名称，文件名称
  - 一般情况下全为小写字母，单词与单词之间用`-`中划线进行连接,不能有空格。`eg: tools my-habits`
  - 在前端的项目代码中： react 组件的文件夹/文件，以大写字母开头。`eg: PascalCase`
  - 特殊情况：某些说明文件的文件名，可以使用大写字母，比如 README、LICENSE。

**函数命名规则：**

一般采用小驼峰命名，构造函数采用大驼峰命名.`eg: camelCase PascalCase`。
